### Как запускать?

Чтобы приложенние работало, нужно запустить фронт и бэк одновременно

##### Бэк:
###### 5000 порт
`npm run server`
Если дропает ошибку `(node:8844)`, нужно добавить IP в Network Access монги



##### Фронт:
###### 3000 порт
`npm start`
